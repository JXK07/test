# 最大厚度 C1+G2 虚拟厚度 BP3333 实现说明

本文件夹实现 `de_virtual_C1_mtC1` 中的新版本。核心变化是：

- 前缘椭圆与前缘 Bezier 在切点处只强制 `C0+C1`。
- 尾缘椭圆与尾缘 Bezier 在切点处只强制 `C0+C1`。
- 最大厚度点处不再强制参数二阶导数完全相等，只保留切向 `C1` 和几何曲率 `G2` 连续。
- `y0_te` 不再由尾缘方程反解，而是作为直接估计参数和优化变量。

## 1. 控制点形式

前缘厚度 Bezier 使用 4 个控制点：

```text
P0 = (0, y0_le)
P1 = (x1, y1)
P2 = (x2, y_t)
P3 = (x_t, y_t)
```

尾缘厚度 Bezier 使用 4 个控制点：

```text
Q0 = (x_t, y_t)
Q1 = (2*x_t - x2, y_t)
Q2 = (xi, y1)
Q3 = (1, y0_te)
```

其中 `Q1=(2*x_t-x2, y_t)` 保证最大厚度点处前后段一阶导数方向和大小一致，也就是参数 `C1`。`Q2.y=y1` 使两段在最大厚度点处的几何曲率相等，因此满足 `G2`。注意本版没有再使用旧版的

```text
xi = x1 - 4*x2 + 4*x_t
```

因为这个式子来自参数 `C2` 的 x 二阶导数连续。当前代码只要求 `G2`，所以 `xi` 由尾缘切点的 x 位置方程确定。

## 2. 前缘方程

给定前缘椭圆切点：

```text
S = (x_s, y_s)
m_s = dy/dx at S
```

并给定 `x_t, y_t, y0_le` 后，代码把未知量化为 `u_s` 的函数。设三次 Bezier 基函数为：

```text
A = (1-u)^3
B = 3*u*(1-u)^2
C = 3*u^2*(1-u)
D = u^3
```

前缘切点 `C0` 与 `C1` 方程为：

```text
x_s = A*0 + B*x1 + C*x2 + D*x_t
y_s = A*y0_le + B*y1 + C*y_t + D*y_t
dy/du - m_s*dx/du = 0
```

代码在 `model.py::solve_virtual_c1_controls()` 中解析消元得到：

```text
x2 = [
    m_s*(x_s*(3*u_s-1) - 2*u_s^3*x_t)
    + u_s^3*(y0_le-y_t)
    + 3*u_s^2*(y_t-y0_le)
    + 3*u_s*(y0_le-y_s)
    + (y_s-y0_le)
] / [3*m_s*u_s^2*(1-u_s)]

x1 = (x_s - C*x2 - D*x_t) / B
y1 = (y_s - A*y0_le - (C+D)*y_t) / B
```

因此前缘没有再求切点曲率连续，`leading_curvature_jump` 只作为诊断输出。

## 3. 最大厚度处 C1+G2

最大厚度点为 `(x_t, y_t)`。前缘段末端导数为：

```text
r'_P(1) = 3*(P3-P2) = (3*(x_t-x2), 0)
```

尾缘段起点导数为：

```text
r'_Q(0) = 3*(Q1-Q0)
```

令 `Q1=(2*x_t-x2, y_t)` 后：

```text
r'_Q(0) = (3*(x_t-x2), 0) = r'_P(1)
```

所以最大厚度处满足参数 `C1`。

对水平切线的端点，曲率只与控制点纵向二阶差分有关。前段末端曲率与后段起点曲率相等要求：

```text
y1 - y_t = Q2.y - y_t
```

因此代码取：

```text
Q2.y = y1
```

对应的最大厚度曲率诊断为：

```text
kappa_t_star = 2*(y1-y_t) / (3*(x_t-x2)^2)
```

代码要求 `kappa_t_star < 0`，即最大厚度点附近曲线向下弯。

## 4. 尾缘方程与 y0_te

尾缘椭圆切点为：

```text
R = (x_r, y_r)
m_r = dy/dx at R
```

`y0_te` 是变量。给定 `u_s` 后，前缘已经确定 `x2,y1`，于是尾缘控制点中只有 `xi` 和 `v_r` 需要由尾缘切点关系确定。

对尾缘 Bezier，在参数 `v_r` 处：

```text
x_r = A*x_t + B*(2*x_t-x2) + C*xi + D*1
```

代码由此解析得到：

```text
xi = (x_r - A*x_t - B*(2*x_t-x2) - D) / C
```

剩下两个方程用于求 `u_s, v_r`：

```text
y(v_r) - y_r = 0
dy/du(v_r) - m_r*dx/du(v_r) = 0
```

这就是当前版本的闭合方程。它没有再用尾缘切点曲率连续，所以 `trailing_curvature_jump` 只作为诊断输出。

## 5. 优化变量

优化变量定义在 `optimization.py::VARIABLE_NAMES`：

```text
x_t, y_t, beta_te, gamma_le,
x_c, y_c, k_c, alpha_te,
z_te, dz_te,
ellipse_a, axis_ratio, theta,
y0_le, y0_te,
te_ellipse_a, te_axis_ratio, te_theta
```

其中：

- `ellipse_b = ellipse_a / axis_ratio`
- `te_ellipse_b = te_ellipse_a / te_axis_ratio`
- `theta` 控制前缘椭圆切点
- `te_theta` 控制尾缘椭圆切点
- `y0_le` 是前缘虚拟厚度
- `y0_te` 是尾缘虚拟厚度，本版会被 SLSQP/GA 优化

## 6. 边界与约束

`build_bounds()` 采用局部优化边界：

- `x_t, y_t` 限制在直接估计值附近，避免最大厚度位置漂移到非物理解。
- `theta, te_theta` 限制在 `(0.03, 1.45)`，避免椭圆端点退化。
- `ellipse_a`、`te_ellipse_a` 和轴比允许较宽范围，以便前后椭圆调整形状。
- `y0_le` 约束在 `(0, 0.95*y_t)` 内。
- `y0_te` 下界为 `0`，上界取 `min(0.95*y_t, 3*max(y0_te_seed,dz_te))`，这样钝尾缘可以增厚，尖尾缘不会无约束膨胀。
- 钝尾缘时 `dz_te` 的上界参考原始尾缘附近厚度，避免再次把厚尾缘压成闭合尖尾缘。

`build_constraints()` 的几何约束包括：

- 前缘和尾缘厚度控制点 x 坐标单调。
- `P`、`Q` 控制点之间保留最小间距。
- `y_t > y0_le` 且 `y_t > y0_te`。
- 尾缘椭圆切点位于 `Q2` 与真实尾缘之间。
- 尾缘椭圆切点厚度为正，且低于 `Q2.y`。
- 弯度线控制点 x 坐标单调。

此外，控制点求解内部还会检查采样 Bezier 段厚度非负、x 单调、`kappa_t_star<0`。

尾缘连续性特别处理：

- 代码在 `fit.py` 中定义 `CONNECTION_RESIDUAL_TOL = 3e-3`。
- 直接估计会优先选择尾缘切点方程残差低于该阈值的候选。
- 优化目标和约束会拒绝残差高于该阈值的候选。
- 这样可以避免出现 MAE 较低但尾缘 Bezier 与椭圆明显跳跃的不连续假解。
- `uvblade.s1` 之前的断裂结果残差约为 `5e-2`，现在会被拒绝；修正后的结果残差约为机器精度。

## 7. 绘图与输出

`plotting.py` 中原始叶型轮廓使用黑色点划线 `-.`，直接估计使用绿色虚线，优化后使用蓝色实线。优化图会同时显示：

- 厚度分布：参考厚度、直接估计、优化结果、前缘椭圆、尾缘椭圆、控制点。
- 叶型轮廓：原始叶型点划线、直接估计、优化结果。

每个叶型会输出：

- `*_bp3333_virtual_mtG2.dat`
- `*_bp3333_virtual_mtG2.json`
- 优化时额外输出 `*_direct.*`、`*_optimized.*` 和 `*_optimization.png`

JSON 中 `parameters.y0_te` 是最终参数值；优化 JSON 中 `optimization.variables.y0_te` 会记录初值、优化值和上下界。

## 8. 常用命令

直接估计并保存图：

```bash
python3 BP3333_ellipse_virtual_C1_mtC2/main.py --airfoil "BP3333_ellipse_virtual_C1_mtC2/Test Airfoils/n0012.dat" --theta 0.65 --y0-le-ratio 0.25 --y0-te-ratio 0.05 --no-show
```

SLSQP 优化并保存直接/优化对比图：

```bash
python3 BP3333_ellipse_virtual_C1_mtC2/main.py --airfoil "BP3333_ellipse_virtual_C1_mtC2/Test Airfoils/uvblade.s1" --theta 0.65 --y0-le-ratio 0.25 --y0-te-ratio 0.05 --optimize --maxiter 60 --no-show
```

GA 快速试算：

```bash
python3 BP3333_ellipse_virtual_C1_mtC2/main.py --airfoil "BP3333_ellipse_virtual_C1_mtC2/Test Airfoils/uvblade.s1" --optimizer ga --ga-population 32 --ga-generations 20 --optimize --no-show
```

批量直接筛查：

```bash
python3 BP3333_ellipse_virtual_C1_mtC2/main.py --all --theta 0.65 --y0-le-ratio 0.25 --y0-te-ratio 0.05 --no-plot --no-show
```
